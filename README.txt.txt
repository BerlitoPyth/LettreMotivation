Installation et lancement de la Lettre de Motivation Interactive

1. Installer Python :
   - Aller sur python.org/downloads
   - Télécharger et installer la dernière version de Python
   - IMPORTANT : Cocher "Add Python to PATH" pendant l'installation

2. Installation des dépendances :
   - Double-cliquer sur install.bat
   - Attendre la fin de l'installation

3. Lancer l'application :
   - Double-cliquer sur run.bat
   - L'application s'ouvrira dans votre navigateur par défaut

En cas de problème :
- Vérifier que Python est bien installé en ouvrant un terminal et en tapant : python --version
- Vérifier que tous les fichiers sont dans le même dossier
- Réessayer l'installation des dépendances

Pour fermer l'application :
- Fermer la fenêtre du navigateur
- Dans la fenêtre de terminal, appuyer sur Ctrl+C
